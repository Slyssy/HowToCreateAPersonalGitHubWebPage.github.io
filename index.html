Create a Personal Page on GitHub

Step1: Create a GitHub Account

  1. Log onto GitHub by going to www.github.com
  2. Fill in the Username, Email and Password.
  3. Click "Sign Up".

Step2: Create a New Repository

  1. Click on the green box that says "New" in the repositories section.
  2. This should take you to a page that looks like this.

Step 3: Name Your Repository

  1. Enter a repository name.
  2. When naming your repository, use the following syntax. [yourusername].github.io
  3. You should always include a ReadMe file that explains the contents of your repository.
  4. It's also a good idea to create a .getignore file by selecting the check box.
  5. Click [Create Repository], and you should get a screen like this.

Step 4: Create Files and Open VSCode

  1. Back in your terminal create a new index.html file by typing touch index.html. (Note: Make sure you are in the correct file in your terminal)
  2. Now open the folder you just cloned in VS Code. Your VS Code screen should look something like this.

Step 5: Upload Changes to GitHub

  1. You'll need to open your terminal. VS Code makes this easy. While in VSCode simply hold down <>[control] + `
  2. Now that your terminal is open, type git status in your terminal. Your terminal should look something like this.
  3. Now type git add .
  4. This adds the files to the queue to be committed. To confirm the files were added to the queue, type git status. Your terminal should look like this.

Step 6: Commit & Push Changes to GitHub

  1. Next, you need to commit the files. This is done by typing git commit -m"Initial Commit". Your screen should look like this.
  2. Finally, we need to push the changes to the GitHub repository. This is done by typing git push.

    3. Your terminal should now look something like this...

Step 7: Commit & Push Changes to GitHub

  1. Check your GitHub repository to verify the changes were uploaded. Your repository should look something like this...

  2. You are DONE! Go to your browser and type in the name you created back in step 2 to access your deployed GitHub personal page.

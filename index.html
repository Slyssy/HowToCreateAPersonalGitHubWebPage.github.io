Create a Personal Page on GitHub

Step1: Create a GitHub Account

  1. Log onto GitHub by going to www.github.com
  2. Create a GitHub account by filling out the Username, Email and Password.
  3. Click "Sign Up".

Step2: Create a New Repository

  1. Click on the green box that says "New" in the repositories section. This should take you to a page that looks like this.
  2. Enter a repository name.
  3. When naming your repository, use the following syntax. [yourusername].github.io

Note: You should always include a ReadMe file, and if you want to add a gitignore file you can select the checkbox. If you create .gitignore file, you will need to select a template. After you select your option click "Create repository". That will take you to a page like this.

Step3: Clone Your New Repository to Your Local Machine

  1. At this point, you simply need to select the green "Code" button. That will pop up a dialogue box like this.
  2. Copy the SSH Link.
  3. Now you will need to open your terminal and navigate to where you want your directory on your local machine.
  4. In your terminal, type git clone [paste the ssh link you copied] and press "Enter". Your terminal should look like this.
